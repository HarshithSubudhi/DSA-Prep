# Day 1: Vector (C++ STL)

## What is Vector?
- Dynamic array in C++
- Automatically resizes
- Part of STL (Standard Template Library)

## Why use Vector?
- No fixed size like arrays
- Easy insertion and deletion
- Widely used in DSA problems

## Important Functions
- push_back(x) → add element
- pop_back() → remove last element
- size() → number of elements
- sort(begin, end)
- reverse(begin, end)

## Code Concepts Practiced
- Traversing vector
- Sorting vector
- Reversing vector
- Finding max element
- Counting elements

## Problems Practiced
1. Reverse array
2. Find maximum element
3. Count even numbers
4. Sort array

## Key Learning
Vector replaces arrays in most DSA problems.
